# Easy ETL

## Description

Easy ETL is a lightweight PHP framework designed to move data from A to B without the overhead of enterprise bloat. Built for developers who need a reliable, "set-it-and-forget-it" pipeline in minutes. This framework was built for HubSpot but you can use it for any other software (Pipedrive, Jira, Aircall, etc.).
